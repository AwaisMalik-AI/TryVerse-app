# TryVerse User App Spec
 
## Goal
Build a mobile-first TryVerse user app for shoppers. This is not a brand dashboard and not a desktop website.
 
TryVerse helps users:
- Try clothes on themselves
- Ask Stylo for fashion advice
- Browse clothing
- Save and compare looks
- Manage profile and settings
 
## Design System
Use the current TryVerse branding:
- Dark premium UI
- Black / deep navy / deep purple background
- Purple, violet, magenta glow accents
- Glassmorphism cards
- Purple gradient buttons
- Thin purple borders
- Soft glow effects
- Rounded modern mobile UI
 
Fonts:
- Headings: Clash Display Regular
- Body/buttons/forms: Montserrat
 
Image rules:
- Clothing/apparel only
- No shoes, bags, jewelry, perfume, beauty, furniture
- No vulgar or revealing images
- Use modest professional fashion visuals
 
## Mobile First
Primary viewport:
- 390 x 844
 
Also support:
- 375 x 812
- 430 x 932
 
Rules:
- This must look like a real mobile app
- No desktop website layout
- No large empty horizontal space
- No desktop navbar on mobile
- Use bottom navigation after login
